# 09 · Data Management Plan

## Data Description

This research will collect quantitative and qualitative data related to municipal solid waste management among households participating in the source segregation program in Amarilis District, Huánuco, Peru.

The collected data will include:

- Household demographic information
- QR Code registration records
- Waste segregation records
- Survey responses
- Artificial Intelligence analysis results
- System-generated reports

---

## Data Storage

All research data will be securely stored in:

- GitHub Repository (documentation)
- Google Drive (backup)
- External encrypted storage
- Local secured computer

---

## Data Security

The following measures will be implemented:

- Password-protected devices
- Regular backups
- Restricted access
- Data encryption where applicable

---

## Data Privacy

Participants' identities will remain confidential.

Personal identifiers will not be included in public datasets.

---

## Data Sharing

Only anonymized research data may be shared for scientific purposes following institutional ethical approval.

---

## Data Preservation

Research data will be preserved throughout the doctoral project and archived after project completion according to university policies.
